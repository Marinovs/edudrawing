# Edu Drawing: your school sketch!

> Tecnologie per il Web: University project.

## Installation

```sh
cd client/
npm install

cd server/
npm install
```

## Usage

To run Front-End:

```sh
cd client/
npm start
```

To run Back-end:

```sh
cd server/
npm run dev
```
